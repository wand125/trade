# Decisions

重要な仕様変更や設計判断を ADR 形式で残す。

## 形式

```text
# NNNN: タイトル

日付:
状態: proposed / accepted / superseded

## 背景

## 決定

## 影響

## 代替案
```

## 一覧

- `0001_documentation_structure.md`: ドキュメント構造
- `0002_multitask_targets.md`: 3クラス分類だけを主ターゲットにしない
- `0003_overfitting_and_walk_forward.md`: データ増加とwalk-forward評価
- `0004_relaxed_adjusted_pnl.md`: 評価倍率 1.0 / 1.25
- `0005_dense_entry_quality_targets.md`: entry timingを密な品質ターゲットに分解
